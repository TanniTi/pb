Цель проекта: закрепить знания ООП на примере игры "Змейка". А также ознакомиться с многопоточностю на примере портала, сквозь который "проползает" змейка.

Чтобы поменять сложность, нужно:
1) перейтив в пункт меню "SETTINGS"
2) клавишей "->" выбрать сложность
3) нажать Esc для игры

В папке "screenshot" содержутся скриншоты основного функционала.

Приятной игры!
